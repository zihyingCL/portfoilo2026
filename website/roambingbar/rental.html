Wi-Fi 設備使用說明

Wi-Fi 連接方式
  1. 請打開手機 Wi-Fi 功能
  2. 搜尋機器背面之「SSID」的 Wi-Fi 名稱
  3. 輸入 SSID 下方之 WPA Key 密碼
設備規格基本說明
  • 電信業者：依各國電信服務供應商有所不同
  • 通訊規格：4G LTE/3G
  • 最多連接台數：5 台
  • 充電時間：8 小時
  • 連續使用時間：15 小時（以 1 台裝置為例）
  • 流量：1GB，超量降速，仍可使用，並於所在國家的首都時間凌晨 00:00 更新流量與恢復速度。
  • 特徵：
    1. 世界通用機種，於該國開機時將會自動連結當地訊號。
    2. 因地區、地形及建築物...等因素，收訊將有所差異。
貼心小叮嚀
  1. 因【國際數位通信公平利用政策】，請避免短時間大量觀看影片、玩遊戲或是視訊通話等短時間高耗流量的使用方式，電信商會維護資訊安全，可能該線路會有速度受限之情事。如因此情事無法正常使用，恕無任何賠償措施。
  2. 本機處於充電或長時間使用狀態時，會有發熱情形之產生。
  3. 出發/歸還前請確認下列物品完整性：
    1. Wi-Fi 分享器
    2. 機器之包包
    3. USB 傳輸線及插頭
    4. 機器及配件相關說明書
